  1. start -> Run
  2. enter `regedit'
  3. in Registry Editor, find HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Internet Explorer
  4. right click on `Internet Explorer', click `New' -> `Key'. enter `View Source Editor'
  5. right click on `View Source Editor', click `New' -> `Key'. enter `Editor Name'
  6. modify the default value (can use double click on `(Default)'). enter the editor path you want, e.g. C:\Program Files\EmEditor\emedhtml.exe

如果是輸出成.reg檔，如下 

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Internet Explorer\View Source Editor\Editor Name]
@="C:\\Program Files\\EmEditor\\emedhtml.exe"
