日文字型的安裝

目前網路上流傳的日文字型主要有中國海字集和櫻花輸入法 兩種，以下就二者作一比較，讓大家自行決定使用那一種：

　 中國海字集 櫻花輸入法
壓縮檔大小 1.43Mb ARJ解354Kb、ZIP解413Kb
解壓縮後大小 2.81Mb 7.64Mb(真是太神奇了)
安裝方法 利用造字程式 同左
安裝說明 十分"簡要" 圖文並茂，說明檔將近7Mb
特殊符號 Yes Yes
日文漢字 Yes No
字體美觀度 這個就見仁見智了

從上表可以大約瞭解兩者的差異，其中櫻花輸入法的檔案大小前後差異會如此大， 是因為它的說明檔是一個將近7Mb的Word文件，神奇的壓縮把7Mb變成200Kb，所以才出現這個情況，而去除說明檔 所剩的檔案大小為752Kb，這還包括了輸入法的部分，所以它本身的字型不如中國海字集多。

如果要我推薦一個的話，我會建議對視窗系統不熟的人用櫻花輸入法， 因為它的說明檔十分詳細，能夠很容易地安裝（我認為:)），而中國海字集 則可和日文假名輸入法合用，日文假名輸入法可以用選單的方式來輸入 日文漢字（前提是有裝中國海字集），而且內附的說明網頁也介紹了中國海字集的安裝，比中國海字集本 身的說明寫得好。

P.S. 櫻花輸入法是包含了日文字型和輸入法， 中國海字集是純字型集，日文假名輸入法只有 輸入法（我覺得這一個比較好用）。
