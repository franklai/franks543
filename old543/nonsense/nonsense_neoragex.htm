您現在所在的位置是：亂寫一通/NeoRAGEx使用說明

　　NeoRAGEx需要主程式和BIOS兩個壓縮檔，如果沒有的話請到這一頁去下載，如果都有了的話，就可以開始玩了， 等一下，只有模擬器沒有ROM玩啥，請看NEOGEO使用方法， 裡面有寫如何處理ROM的方法。一切準備就緒，進入NeoRAGEx吧

　　上面的圖就是NeoRAGEx的主畫面，左上方的GAMELIST是遊戲列表， 如果有偵測到遊戲的話，遊戲名稱會變成白色。左下方是預覽，因為空間有限， 所以本站提供的NeoRAGEx是沒有預覽的。
　　右下方是事件，在其上方的三個按鈕分別是發射、輸入、離開。

  • 在遊戲列表點選好要執行的遊戲，再按下發射，NeoRAGEx就會開始執行了。
  • 當ROM已經放在指定的資料夾，而遊戲列表沒有出現遊戲的話，就要按輸入這個鈕了。
  • 離開，這個不用多做介紹了吧...

    OPTIONS畫面

  • CONTROLS：就是設定搖桿、鍵盤的地方。
  • DIRECTORIES：設定放遊戲的目錄以及其他目錄。
  • VIDEO OPTIONS：畫面影像的設定。
  • AUDIO OPTIONS：聲音的設定。
  • MACHINE SETTINGS：機器設定？
  • SHOTS FACTORY：平時是灰色的，功用是抓圖，後面會有解說。

    DIRECTORIES畫面

  • ROMPATH：遊戲位置，\是指和NeoRAGEx主程式同資料夾。
  • PREVIEW PATH：預覽位置。
  • SHOTS PATH：抓下來的圖位置
  • STATE PATH：???
  • HISCORE PATH：最高分紀錄位置
  • WAVES PATH：抓下來的聲音位置

    VIDEO OPTIONS畫面

  • COLOR DEPTH：顏色深度？有16BIT/8BIT兩種選項，選擇8BIT時，INTERPOLATION無法使用
  • SCANLINES：掃瞄線，模擬電視機會有的線條，有YES/NO/50%三種選項，當COLOR DEPTH選擇8BIT時，沒有50%可選
  • INTERPOLATION也不知道是什麼，選YES後畫面比較矇矓
  • EDGE ENHANCEMENT邊緣修飾，讓畫面看起來比較圓滑
  • TRIPPLE BUFFERING不知道，選NO時VSYNC無法使用
  • FRAMESKIP頁框略過？有AUTO/0~11可選
  • VSYNC還是不知道，有YES/NO可選

    AUDIO OPTIONS畫面

  • MAIN VOLUME：就是聲音大小，往右邊拉大聲，往左小聲
  • SOUND：選擇只有音效/只有音樂/兩者都有
  • OUTPUT CHANNELS：以STEREO或MONO來播放
  • SAMPLE SIZE：???
  • SAMPLE RATE：???
  • INTERPOLATED SOUND：???

    MACHINE SETTINGS畫面

  • LANGUAGE：選擇語系，有US/JAPAN/EUROPEAN可選
  • SYSTEM：選擇家用CONSOLE/大型電玩ARCADE
  • MVS SETTINGS：大型主機設定是否開啟
  • FREEPLAY：大型主機免投幣是否開啟

    SHOT FACTORY

    SHOT FACTORY在平時是灰色的，無法點選，使用時機是在進入遊戲後， 按Tab鍵，選SHOT FACTORY，就會出現SHOT FACTORY的編輯視窗， 抓下來的圖片會放在DIRECTORIES設定的SHOTS PATH裡。
  • OBJECT：每一張圖的OBJECT數量都不一樣，把╳取消掉，會有一些東西不見
  • MAKE ALL：OBJECT全部打╳，代表整張畫面都擷取
  • CLEAR ALL：OBJECT全部不打╳，
  • FRONT LAYER：把╳取消掉，字會不見(例如下方的CREDITS)
  • ALT. BACKCOLOR：???
  • NEXT FRAME：???
  • SAVE SHOT：就是把圖給儲存起來

    Tab鍵的使用

    除了抓圖之外，Tab鍵還有一些其他的用途，例如：當已經進入遊戲後，才想到要改按鍵設定， 如果跳出遊戲來解決的話，再進入遊戲又得等上一會兒，這時候只要按Tab鍵跳到設定畫面， 就可以改要改的設定了！
    (注意！！如果要改MACHINE SETTINGS的設定，改完後必須按RESET才會作用。)

    如何無限接關

    　　Frank功力太差，正常地玩越南大戰沒幾關就GAME OVER了，可是Frank還是有辦法玩到全破， 不要以為是用修改大師，Frank太窮了買不起，以下要介紹的就是讓再肉腳的人也能全破的方法－ 無限接關
    　　首先要改MACHINE SETTINGS，把SYSTEM選成ARCADE，FREE PLAY選成ON， 然後再進入遊戲，您就會發現不論死了多少次還是能接關。
    如果您使用上述方法後不能無限接關，可能是BIOS的關係， 請下載本站的BIOS 。

    熱鍵

    　　雖然遊戲進行到一半時想改設定的話，只要按Tab鍵跳出來改就好，但是有人還是覺得太慢了， 所以NeoRAGEx有所謂"熱鍵"的設計， 以下就是熱鍵用,[]代表解釋,{}內為原文：
  • TAB：切換進/出系統選單[就是上面講的設定畫面]{Toggle in/out to system menu}
  • 1：玩家1的開始鍵{start 1 player game}
  • 2：玩家2的開始鍵{start 2 player game}
  • 3：玩家1的投幣鍵{Coin1/Select}
  • 4：玩家2的投幣鍵{Coin2/select}
  • 5：聲音調小聲{Volume - }
  • 6：聲音調大聲{Volume + }
  • 7：記錄聲音輸出{Record sound output}
  • 8：顯示/隱藏 頁框/每秒{Show/Hide frames/Sec}
  • 9：全速？(開/關){Full throttle (ON/OFF)}
  • 0：儲存預覽圖片(只有8-bit下可行){Save preview shot (8-bit only)}
  • F1：Reset或切換進/出MVS設定[如果MVS SETTINGS選ON的話]
    　　　{Reset (Toggle in/out to MVS-settings)}
  • F2：切換聲音設定(音效與音樂/音樂/音效/都沒有){Audio setting (ADPCM/FM/OFF)}
  • F3：聲音8/16-bit切換{8/16-bit audio output}
  • F4：聲音頻道(普通/立體){Audio channels (MONO/STEREO)}
  • F5：8/16-bit顏色深度切換?{8/16-bit colordepth}
  • F6：掃瞄線(開/50%/關){Scanlines (ON/50%/OFF)}
  • F7：畫面模糊?{Interpolation (ON/OFF)}
  • F8：邊緣修飾(開/關){Edge enhancement (ON/OFF)}
  • F9：抓圖鍵{Take Screenshot}
  • F11：儲存{Save state menu}
  • F12：載入{Load state menu}
  • ESC：嗯...這個鍵能做什麼？{Hmmmm... shat might this key do?}
