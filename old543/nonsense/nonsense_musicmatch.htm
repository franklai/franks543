您現在所在的位置是：亂寫一通/如何製作MP3檔案

　　不多說廢話，直接來看如何製作MP3吧。這裡所用的軟體是 MusicMatch JukeBox 5.00.171，這套軟體可以在軟體下載裡面有。

　　將軟體安裝好後執行（好像是廢話），在上方選單中選取 Options/Recorder/Setting，由於小弟才疏學淺，所以只知道幾個設定， 其它的就請看倌另尋高明了。
　　首先，在Recording Quality這個大框裡選擇 要錄製的格式，這次的主題是MP3，所以選擇MP3(128kbps)，而 Windows Media就是最近也蠻熱門的wma格式，Wav Format則是 複製成好幾十MB的wav檔，用途是燒CD用的（好像偏離主題了）， 下方的Recording Source是選擇錄音來源，這個就要選您的CD-Rom代號， 就是除了Line In、Mic In、System Mixer剩下那一個（除非您有多個CD-Rom）。 Mute while Recording沒打勾時，會一邊錄一邊放，和用音響錄是一樣的， 錄音的時間和歌曲的時間一樣，建議勾取，這樣速度會比較快。 再來按一下Song Directory，Directory For New Songs是您放置MP3檔案的資料夾， 而到時候存放的路徑就是下面Sample Path寫的一樣，其它Make Sub-Path using是在資料夾 下再加資料夾，可以加歌手名稱或（和）專輯名稱，Name Song File using 是指檔案的名稱，有四項可以選，Separator是各名稱連結時用的符號。
　　設定結束後，把CD放進CD-Rom，在Recorder部份勾取要錄的歌 （如果沒有Recorder部份，在上方選單把 Options/View/Show Recorder打勾）， 再來按下圓圓的REC就會開始錄製了，等到百分比條跑完您的MP3也完成了。
